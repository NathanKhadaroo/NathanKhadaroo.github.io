How to run the code

To run the model the files must be in the same directory and the graphics backend for the IPython console should be set to "Automatic".

The model is run via the input file ("Run_this_file.py"), which prompts the user to enter values for the parameters of the model and then runs the model.

More detail about how the model works can be found in the readme file.
